# A skeleton Vue + Tailwind CSS project

### Getting Start
```
$ git clone git@github.com:horaceho/vue-tailwind-skeleton.git skeleton
$ cd skeleton
$ yarn install
$ yarn serve

$ open http://localhost:8080/
```
